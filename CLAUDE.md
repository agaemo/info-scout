# info-scout 開発ガイド

## 概要

毎日 Web 検索・収集し Slack に通知する汎用情報収集エージェント。

## 開発フロー

- 変更は必ず Issue を作成してから着手する
- 実装は feature ブランチで行い、PR を作成してマージする
- `main` への直接コミットは禁止

## トピックの追加・変更

`topics.json` を編集する。構造：

```json
[
  {
    "name": "トピック名",
    "queries": ["検索クエリ1", "検索クエリ2"]
  }
]
```

## 動作確認

```bash
./run.sh
```

## 環境変数

`.env` に `SLACK_WEBHOOK_URL` を設定する（`.env` はコミットしない）。
